# Team Bridge

## Projeto de mestrado de Alan Klinger
## UFRN - IMD - Instituto Metrópole Digital


O TeamBridge é um middleware que utiliza o VRPN para facilitar a utilização de jogos terapêuticos com dispositivos de entrada.

A [Wiki](https://github.com/klingerkrieg/Team-Bridge/wiki/) deste repositório conta com instruções para sua utilização.
